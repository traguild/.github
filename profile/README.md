## 프로젝트 개요
- **프로젝트 명:** 지역 의뢰 플랫폼
- **프로젝트 기간:** 2024.09 ~ 2025.05 (약 8개월)
- **프로젝트 목표:**
  - 모바일 앱을 통하여 간단한 의뢰(심부름, 잡일 등)을 간편하게 맡기고 수행
  - 지도의 마커를 통하여 자신 주변에 있는 의뢰들을 쉽게 확인
  - 완료된 의뢰에 따라서 경험치와 칭호를 부여하는 것으로 사용자의 재미 요소를 추가
